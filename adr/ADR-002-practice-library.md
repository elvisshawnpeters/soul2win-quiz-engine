# ADR-002: Practice Library

Status: Accepted
Date: 2026-07-21

## Context

Students who miss a daily quiz cannot access previous assessments easily.

## Decision

Introduce a dedicated Practice Library page instead of adding a dropdown to the home page.

## Rationale

- Keeps the home page focused on today's practice.
- Scales to 40+ assessments.
- Enables future search and filtering.
- Provides a better experience for revision.

## Consequences

- Requires `tests.json`.
- Enables future Progress Dashboard.
