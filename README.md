# soul2win Quiz Engine

Version 1.0

A lightweight, reusable Quiz Engine built using HTML, CSS and Vanilla JavaScript.

Designed to power educational quizzes such as NEET, Bible, GK, school assessments and future AI-generated practice tests.

## Features

- Responsive Design
- Multiple Choice Questions
- Instant Scoring
- Previous / Next Navigation
- Question Bank Separation
- Easy to Customize
